# Communication & Planning Rules (커뮤니케이션 및 플래닝 규칙)

이 문서는 AI 어시스턴트가 사용자와 소통하고 작업을 계획할 때 준수해야 할 규칙을 정의합니다.

## 1. Language Preference (언어 설정)
- **Principal Language**: 모든 대화와 응답은 한국어를 사용하여 진행합니다.
- **Technical Terms**: 코드, 변수명, 기술적 전문 용어, 라이브러리 이름 및 에러 메시지는 의미 전달을 위해 영문을 그대로 사용할 수 있습니다.

## 2. Planning (플래닝)
- **Korean Plans Mandatory**: 코드 수정이나 복잡한 작업을 수행하기 전, AI가 제시하는 **수행 계획(Plan)은 반드시 한국어로 작성**합니다.
- **Structure**: 단계별로 명확하게 나누어 사용자가 작업 흐름을 쉽게 이해할 수 있도록 구성합니다.
